# Data Overview: Sleep Deprivation, Cognition, and Emotion

This document provides an overview of the dataset used in this project, including study design, measurement domains, and research focus. The purpose is to establish a clear understanding of the data before preprocessing and analysis.

## Dataset Description

The dataset consists of behavioral and self-report data collected from 60 participants as part of a 2024 study examining the effects of sleep deprivation on cognitive performance and emotional regulation.

Measures include sleep duration, sleep quality, daytime sleepiness, cognitive task performance, emotional stability, and demographic and lifestyle variables such as age, gender, BMI, caffeine intake, physical activity, and stress levels.

## Cognitive Tasks Included

- Stroop Task: Cognitive control and response inhibition
- N-Back Task : Working memory performance
- Psychomotor Vigilance Task (PVT): Sustained attention and vigilance

## Research Focus

This project examines how sleep deprivation affects cognitive control, working memory, and vigilance, and how these effects relate to emotional regulation.
