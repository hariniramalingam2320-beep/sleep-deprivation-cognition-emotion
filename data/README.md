## Dataset

The dataset used in this project was obtained from Kaggle and is not redistributed in this repository due to licensing restrictions.

To reproduce the analyses:
1. Download the dataset from Kaggle using the link below.
2. Place the file in the `data/raw/` directory.
3. Rename the file to `sleep_cognition_emotion_raw.csv` (or `.xlsx`).

Kaggle source:
https://www.kaggle.com/datasets/sacramentotechnology/sleep-deprivation-and-cognitive-performance?resource=download
